# iOS Schedule Todo App Prototype

An iOS-style Liquid Glass schedule and todo management app prototype built with Vite, React, and TypeScript.

## Local Development

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:5173/`.

## Build

```bash
npm run build
```

## GitHub Pages

This repository can be deployed to GitHub Pages from the `gh-pages` branch.
